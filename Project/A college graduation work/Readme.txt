This is my graduation work for the 2020-1 semester of Myongji University Nature Campus made of Android Studio.

This app is an application that shows the user and receives feedback when detecting fine dust, gas, and rainwater in real time.

There may still be some errors, and code stabilization and optimization will be done later.
